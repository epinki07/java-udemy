# Java Udemy

Repositorio de practica del curso "Aprende Java con 100 ejercicios practicos (Incluye JavaFX)" de Udemy — 21 horas, mas de 150 ejercicios completados.

Certificado de finalizacion: [UC-e1883c64-c1e4-4f4a-b06f-751a5457dcdd](https://ude.my/UC-e1883c64-c1e4-4f4a-b06f-751a5457dcdd)

## Que cubre

Variables, operadores y control de flujo, arreglos y matrices, programacion orientada a objetos (herencia, polimorfismo, interfaces), excepciones, manejo de archivos, colecciones, Swing, JavaFX, MySQL y JDBC.

## Como correrlo

```bash
git clone https://github.com/epinki07/java-udemy.git
cd java-udemy

# Compilar una seccion
javac -d out src/seccion01_fundamentos/*.java
java -cp out seccion01_fundamentos.Ejercicio01
```

## Tech Stack

Java 11+, Swing, JavaFX, MySQL, JDBC.
