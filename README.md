# 🎓 Java Udemy - Práctica Intensiva

Repositorio de práctica intensiva con más de **150 ejercicios de Java** completados del curso en Udemy.

## 📋 ¿Qué incluye?

Este repositorio documenta mi **volumen de práctica** en Java. Más que un producto final, sirve como respaldo de constancia y dedicación.

## 🚀 Cómo correrlo

```bash
# Clonar
git clone https://github.com/epinki07/java-udemy.git
cd java-udemy

# Compilar sección específica
javac -d out src/seccion01_fundamentos/*.java

# Ejecutar
java -cp out seccion01_fundamentos.Ejercicio01
```
